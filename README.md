# 🐰 CodeBunny

A cute and smart rule-based code reviewer that helps you improve your code with friendly suggestions — no API required 💖

---

## ✨ Features
- 🧠 Rule-based code analysis
- ⚡ Instant feedback
- 🌸 Cute and aesthetic UI
- 💡 Beginner-friendly suggestions
- 🐰 No API / no backend needed

---

## 🎀 How it works
1. Paste your code in the editor
2. Click "Review Code"
3. Get cute + helpful suggestions instantly 💕

---

## 💖 Tech Stack
- HTML
- CSS (Aesthetic UI ✨)
- JavaScript (Rule Engine 🧠)
